Drone Motor

I was the project leader for this drone motor. Through this project, I learned how to develop mass production products. This process involved first completing the mechanical design, creating a golden sample, drafting the assembly SOP, and conducting product validation for the golden sample.

Motor Control Firmware

This is a recently completed project focused on motor control. I successfully developed a sensorless zero-crossing detection control program for a brushless DC motor. The project also involved working closely with the client to define specifications, understand use cases, and action flow, which allowed me to write the program logically. For the mechanical design aspect, I handled some basic sheet metal design tasks.

Water-Cooled Driver Enclosure

This project involved a water-cooled driver enclosure. I proposed a practical implementation plan, including methods for waterproofing and communication. After completing the mechanical design, I proceeded to validate the product.
